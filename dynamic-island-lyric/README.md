# 灵动岛歌词

在桌面顶部显示一个灵动岛式悬浮歌词胶囊，用于轻量查看当前歌词和控制播放。

## 功能

- 独立透明浮窗，默认置顶并记住位置，可放到 Windows 任务栏等工作区外区域。
- 根据 `nowPlaying` 快照订阅播放和歌词，并在本地推算当前歌词行。
- 默认胶囊只显示方形封面、当前歌词和右侧小型律动条，避免内容被控制按钮挤压。
- 展开密度下顶部放置封面、`歌曲名 - 歌手`、居中播放控制、控制区进度条和律动条，歌词区域移到底部横向铺满，可同时显示当前歌词、副歌词和下一句。
- 展开密度下歌词单行展示，超出宽度时横向滚动。
- 支持浮窗右上角透明图钉按钮切换窗口置顶。
- 支持标准、展开两种密度。
- 支持深色、浅色或跟随 EchoMusic 外观。
- 支持空闲淡出和鼠标穿透，适合放在屏幕顶部常驻。

## 要求

- EchoMusic `>=2.2.6-beta.21`
- 主程序需要支持插件浮窗新特性与 `nowPlaying`

## 安装

在线安装：在 EchoMusic 的“插件管理”中添加本仓库插件源，然后安装“灵动岛歌词”。

手动安装：将 `dynamic-island-lyric` 整个文件夹复制到 EchoMusic 插件目录，刷新并启用插件。
