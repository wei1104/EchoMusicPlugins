# 频谱可视化

把 EchoMusic 当前播放音频的频谱嵌入播放器界面，作为轻量背景显示。

## 功能

- 在主窗口 PlayerBar 中显示频谱背景。
- 在 mini 播放器卡片中显示频谱背景。
- 在歌词页底部控制栏上方显示频谱区域。
- 三个显示位置可以分别开关。
- 支持柱状、波形、混合模式，以及刷新率、柱数、FFT、平滑、配色、不透明度等设置。
- 背景光晕独立开关，默认关闭，只保留频谱柱/波形本身。

## 要求

- EchoMusic `>=2.2.6-beta.10`
- 主程序需要支持 `ctx.audio.spectrum`

## 安装

在线安装：在 EchoMusic 的“插件管理”中添加本仓库插件源，然后安装“频谱可视化”。

手动安装：将 `spectrum-visualizer` 整个文件夹复制到 EchoMusic 插件目录，刷新并启用插件。
